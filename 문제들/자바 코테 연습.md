## 스티커 모으기(2)

[코딩테스트 연습 - 스티커 모으기(2)](https://programmers.co.kr/learn/courses/30/lessons/12971)

원판이기에 짝수,홀수 차이가 있다고 생각해서 둘로 나뉘어 코드를 짰다.

결과는 100점 중 6점

- 틀림 1

    ```java
    class Solution {
        public int solution(int sticker[]) {
            int answer = 0;
            int N = sticker.length;
            int total = 0;
            for(int i=0; i<N; i++) {
            	total += sticker[i];
            }
            
            if(N%2 ==0) {
            	int val = 0;
            	for(int i=0; i<N; i+=2) {
            		val += sticker[i];
            	}
            	answer = Math.max(val, total-val);
            }else {
            	
            	for(int i=0; i<N; i++) {
            		int cnt = N/2;
            		int val = 0;
            		int index = i;
            		while(cnt>0) {
            			cnt--;
            			val += sticker[index];
            			index += 2;
            		}
            		if(val > total-val) {
            			answer = val;
            			break;
            		}
            	}
            }
            return answer;
        }
    }
    ```

## 영어 끝말잇기

[코딩테스트 연습 - 영어 끝말잇기](https://programmers.co.kr/learn/courses/30/lessons/12981)

- 정답

    ```java
    import java.util.*;

    public class Solution {
    	public int[] solution(int n, String[] words) {
            int[] answer = {};
            int wordSize = words.length;
            int[] people = new int[n];
            Vector<String> checkWord = new Vector<String>();
            int index =0;
            int nth = -1;
            for(int i=0; i<wordSize; i++) {
            	if(checkWord.contains(words[i])) {
            		people[index%n]++;
            		nth = index % n;
            		break;
            	}else {
            		String tmp = "";
            		if(!checkWord.isEmpty()) {
            			tmp = checkWord.lastElement();
            		}
            		if(!checkWord.isEmpty() && tmp.charAt(tmp.length()-1) != words[i].charAt(0)) {
            			people[index%n]++;
            			nth = index % n;
            			break;
            		}
            		else {
            			people[index%n]++;
            			checkWord.add(words[i]);
            		}
            	}
            	index++;
            }
            answer = new int[2];
            answer[0] = nth+1;
            answer[1] = nth == -1? 0 : people[nth];
            return answer;
        }
    }
    ```
